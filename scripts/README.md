# Scripts Directory

This directory contains utility scripts for the TCG Market application.

## Scraper Loaders
These scripts run specific scrapers and load the data into the database.

-   `load-goblin.ts`: Goblin Store
-   `load-lenocards.ts`: Leno Cards
-   `load-gameland.ts`: Gameland
-   `load-carddunyasi.ts`: Card Dunyasi
-   `load-tizycards.ts`: Tizy Cards
-   `load-cardhouse.ts`: Card House
-   `load-gorilla.ts`: Gorilla Custom Cards
-   `load-mythos.ts`: Mythos Cards
-   `load-overgame.ts`: Overgame

## Utilities
-   `migrate.ts`: Sets up the database schema.
-   `list-remaining-sellers.ts`: Checks which sellers from the Google Sheet are not yet implemented.

## Archive
The `archive/` directory contains old debug scripts and temporary files.
